# 极云终端 
基于Python3开发的终端系统
目前支持的命令:

    cpu  查看cpu核心数
    disk 查看磁盘情况
    scan 扫描固定IP端口
    ip 查看ip
    memory 查看内存使用情况
    times 查看主机开启时间
    install 使用YUM安装程序
    run  运行shell命令
